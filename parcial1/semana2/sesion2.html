DBMS en máquina virtual de Google Cloud

Fecha: 23 de Enero, 2026

1. Objetivo y Contexto

Objetivo: Configurar el DBMS para empezar a usarlo, crear nuevos usuarios desde el administrador y verificar cómo funciona la asignación de los diferentes permisos en la BD.

Contexto: En nuestra máquina virtual previamente teníamos instalado MariaDB, configurado para el arranque automático al iniciar la instancia de la VM.

2. Conceptos y Profundización

Dentro del servidor, con el superusuario configuramos el perfil de administrador del DBMS. Una vez configurado, accedimos como administradores, creamos una base de datos y un usuario con los permisos para alterar dicha base de datos (usando principalmente DDL) y finalmente realizamos un backup de la misma.

  • DDL: Comandos de SQL utilizados para definir, modificar y gestionar la estructura de los objetos en una base de datos.
  • root: El usuario root del servidor Linux tiene la facultad para ejecutar comandos en el sistema operativo, mientras que el root del DBMS tiene la facultad de gestionar todo dentro del gestor de base de datos.
  • DBMS: Database management system*(sistema gestor de bases de datos).
  • Limitaciones: Para este ejercicio guiado, la mayor limitante viene de la poca familiarización con el ambiente de Linux.

Instrucciones rápidas:

$ sudo mariadb_secure_installation
$ su - lenin-campos
$ mysql -u root -p
$ CREATE DATABASE  Biblioteca;
$ GRANT ALL PRIVILEGES ON Biblioteca.* -> To biblio_user@localhost IDENTIFIED BY '666';
$ mysql -u biblio_user -p
$ mysqldump -u biblio_user -p Biblioteca > respaldo_biblioteca.sql
$ more respaldo_biblioteca.sql

5. Conclusiones

Aprendizaje: Se pueden usar simultáneamente varias terminales para crear usuarios con permisos diferentes que usen la BD y realizar respaldos.

Problemas y Soluciones: Al comienzo me atrasé con un comando, pero empecé a comprender lo que estábamos tratando de hacer y llego un punto en que sin necesidad de copiar los comandos, con saber que teniamos que hacer podia hacerlo.

Mejoras Futuras: Practicar más el uso de la terminal.
