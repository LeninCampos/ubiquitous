Configuración de Ubiquitous y Google Cloud

Fecha: 20 de Enero, 2026

1. Objetivo y Contexto

Objetivo: En este día se vieron dos temas principales:
-Comprender cómo acceder y usar Ubiquitous: utilizar nuestro servidor y crear la página donde estaremos subiendo nuestras tareas y ejercicios guiados.
-Configurar nuestra primera máquina virtual en Google Cloud: qué implica, las diferentes opciones y lo que nosotros estaremos usando.

Contexto:
-Todos tenemos una cuenta en el servidor de la escuela; ahí es donde cada quien estará subiendo sus tareas en lugar de Blackboard.
-Previamente dejamos nuestras cuentas escolares de Google Cloud creadas y configurada la facturación.

2. Conceptos y Profundización

  • Ubiquitous: El servidor está previamente configurado con SSH para que cada alumno pueda acceder con su matrícula. Creamos con IA generativa una página web provisional para levantarla en ese servidor y descargamos CyberDuck para poder subir los archivos con seguridad mediante el protocolo SFTP, logrando que se vea una página en vez de un índice en blanco.
  • Google Cloud: Vimos con especial énfasis las configuraciones de las VM, el sistema operativo que debíamos instalar (CentOS), revisamos rápidamente algunos comandos e instalamos MariaDB en el servidor.
  • SSH: Es un protocolo de seguridad de red que permite el acceso y control remoto de equipos a servidores a través de una conexión cifrada.
  • VM: Máquina virtual que sirve para emular el software de una computadora física, funcionando como un sistema independiente con su propio sistema operativo y recursos.
