Descarga de archivos desde internet y editores de texto en terminal

Fecha: 3 de Febrero, 2026

1. Objetivo y Contexto

Objetivo: Importar y manipular archivos sql en los dos diferentes DBMS que estamos, viendo las diferencias e implicaciones de cada uno, así como ejecutar un editor de texto en la terminal como vendría a ser VI, para manipular los archivos desde la terminal, sin necesidad de manipularlo de manera externa.

Contexto: Descargamos la herramienta wget para poder descargar desde internet un archivo sql para manipular con los diferentes DBMS.

2. Conceptos y Profundización

  • wget: Herramienta de línea de comandos del proyecto GNU, libre y no interactivo, diseñada para descargar contenido de servidores web de forma robusta utilizando HTTP, HTTPS y FTP. (GNU Project, 2024)
  • VI: Es un editor de texto de pantalla completa, clásico y predeterminado en la mayoría de los sistemas UNIX/Linux, esencial para la administración de sistemas y emergencias por sus mínimos requisitos de recursos.

&mysqldump --user=biblio_user --password --no-data Biblioteca > Respaldo-Biblioteca.sql

//Solo datos
&mysqldump --user=biblio_user --password --no-create-info Biblioteca > Biblioteca-datos.sql

//Respaldo del DBMS
&mysqldump --user=root --pasword --all-databases > TodasLasbases.sql

//Restaurar un BD
&-u usuario -p
&mysql --user=usuario --pasword MiBase < EsquemaMiDB.sql

//POSTGRES
//Toda la BD
&pg_dumpall -U usuario -f /archivo.sql
&pg_dumpall -U usuario 01 -d MiDB.sql -f RespaldoMiDB.sql

//SOLO DATOS
&pg_dumpall -U usuario01 --data-only MiDB > RespaldoDatos.sql

//Solo esquema
&pg_dumpall -U usuaio01 --schema-only MiDB > EsquemaMiDB.sql

//Restaurar
&psql -U postgres -f /ruta_absoluta/archivo.sql
&psql -U usuario -d /MiDB -f /ruta_absoluta/archivo.sql
&pg_dumpall -f respaldo.sql

Ejercicios:

//Crea un query que muestre una lista de prestamos pendientes, 
//tomando en consideración que la fecha de devolución es NULL
SELECT * 
FROM prestamos 
WHERE fecha_devolucion IS NULL;

//Muestra la cantidad de libros prestados por cada alumno
SELECT a.matricula, a.nombre, COUNT(p.id_prestamo) AS n_prestamos
FROM  alumnos a 
JOIN prestamos p ON a.matricula = p.matricula
GROUP BY a.matricula; 

//Muestra los libros prestados durante el mes de agosto del 2024
SELECT l.titulo
FROM libros l JOIN prestamos p ON l.id_libro = p.id_libro
WHERE p.fecha_prestamo BETWEEN '2024-08-01' AND '2024-08-31';

//Muestra los alumnos que hayan prestado algún libro de "Gabriel Garcia Márquez"
SELECT DISTINCT a.matricula, a.nombre
FROM libros l JOIN prestamos p ON l.id_libro = p.id_libro JOIN alumnos a ON a.matricula = p.matricula
WHERE l.autor = 'Gabriel García Márquez';


//Muestra el listado de libros que nunca hayan sido prestados
SELECT DISTINCT *
FROM libros l
WHERE l.id_libro NOT IN(
	SELECT l.id_libro 
	FROM libros l JOIN prestamos p 
	ON l.id_libro = p.id_libro);

Problemas y Soluciones: El tema de las minúsculas y mayúsculas que es diferente, y con el mismo vi se pudo solucionar rápidamente eso.
